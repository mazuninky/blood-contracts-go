# BloodContracts::Go

## Installation

1. Just go get it:

```sh
$ go get -u github.com/mazuninky/blood-contracts-go
```

2. Import it in your code:

```go
import contracts "github.com/mazuninky/blood-contracts-go"
```

## Refinement Data Type

### Create type

